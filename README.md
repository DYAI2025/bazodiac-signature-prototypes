# Bazodiac Signature Prototypes

This repository contains visual prototypes for the 5D Signature representation, based on the Fusion Coherence Index.

## The Vision
The signature is not a static image, but a visual representation of a 13-dimensional input vector (Natal + Quiz), modulated by cosmic events and personal inputs.

### 1. Gravitational Membrane
A 3D elastic fabric where identity nodes act as masses, curving space. 
- **Harmony:** Smooth basins, fluid flow.
- **Friction:** Sharp peaks, folds, tension.
- **Cosmic Weather:** Subtle vibrations/waves in the fabric.

### 2. Cymatic Resonance Sphere
A sphere of light particles forming geometric structures via frequencies.
- **Harmony:** Crystalline, symmetrical shapes.
- **Friction:** Fractal, organic distortions.
- **Cosmic Weather:** Shifts in base frequency, altering symmetry.

### 3. Bioluminescent Neuro-Nebula
A network of glowing filaments and nodes, similar to a mycelium.
- **Harmony:** Stable, pulsing light-bridges.
- **Friction:** Electric sparks, repellent forces.
- **Cosmic Weather:** Light pulses traveling through the network.

## Implementation Path
Each prototype will be implemented as a standalone Three.js/WebGL experience to evaluate fluidly and aesthetic impact.
